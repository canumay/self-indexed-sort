# Group15 Term Project

Scientific Paper: A New Sort Algorithm: Self-Indexed Sort

### Installation

This repository requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies to run the code.

```sh
$ npm install
```

For development

```sh
$ npm run dev
```

For testing
```sh
$ npm run test
```
